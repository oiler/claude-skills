## How to Update

When making changes:

1. Add to **[Unreleased]** section under appropriate category
2. Use present tense ("Add feature" not "Added feature")
3. Reference issue/PR numbers if applicable
4. Group similar changes together

When releasing:

1. Change `[Unreleased]` to `[Version] - Date`
2. Add new `[Unreleased]` section at top
3. Update version in project.yml
4. Tag the release in git
